# Arbitrager Module — Arbitrage Algorithm Model

## Overview
The Arbitrager module scans multiple exchanges for price discrepancies and constructs arbitrage transaction arrays for execution.

## Key Rule
**No Uniswap integrations**. All arbitrage uses blockchain-only addresses from PostgreSQL.
