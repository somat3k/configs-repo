# DeFi Module — DeFi Services Algorithm Model

## Overview
The DeFi module manages on-chain DeFi service operations using HYPERLIQUID as the primary broker.

## Key Rules
- **NO Uniswap** — strictly forbidden
- HYPERLIQUID is the primary DEX/perpetuals platform
- Minimum two broker fallbacks configured
- All addresses from PostgreSQL only
